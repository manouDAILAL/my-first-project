# Bank Client Management System (C++)

A command-line application for managing bank client data.

Available features:
- Add a new client
- Display client list
- Delete a client
- Update client data (PIN, name, phone, balance)
- Search for a client by account number
- Deposit, withdraw, and view total balances

Data is saved in a text file (`clientsinfo.txt`), with fields separated by `#//#`.
